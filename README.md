# Hello World em Dart

**Aluno:** Higor Assmé Gondin

## Objetivo

Configurar o ambiente de desenvolvimento Dart e executar um programa simples que imprime "Hello World" no terminal, além de praticar o uso de Git e GitHub.

## Como executar

1. Instale o SDK do Dart: https://dart.dev/get-dart
2. Clone o repositório:
   ```bash
   git clone <url-do-repositorio>
   ```
3. Acesse a pasta do projeto:
   ```bash
   cd hellowrld
   ```
4. Execute o programa:
   ```bash
   dart hello_world.dart
   ```

## Saída esperada

```
Hello World
```
